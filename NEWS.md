# MiCT 2.0.0

## Major changes

* Renamed package from `MIC` to `MiCT`.
* Added `sim_threshold()` for LCFA-based threshold estimation for single-item measures.
* Added `mim_threshold()` for LCFA-based threshold estimation for multi-item questionnaires.
* Added `sim_mic_lcfa()` for LCFA-based MIC estimation for single-item measures.
* Updated `mic_lcfa()` with CFA/probit expected-score mapping  
* Added `equalize_levels()`, `var_discretize()` and `lcfa_model()`  
* Renamed `simdat()` output `empirical_rel_trt` to `observed_rel_trt`.
* Updated documentation, examples, and tests.
